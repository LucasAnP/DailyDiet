### Getting Started

Clone the project repository

- [https://github.com/LucasAnP/DailyDiet](https://github.com/LucasAnP/DailyDiet.git)

Access the project directory

```
cd DailyDiet
```

Install the dependencies

```
yarn
  or
npm install
```

Start the APP

```
yarn android
  or
npx expo start

```
